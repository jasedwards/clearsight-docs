Adding a Footer and Actions to Your Widget

Footers and actions are injected by your widget component into the parent WidgetContainerComponent. The easiest way to do this is by extending the base class ActionsAndFooter. Depending on what your widget has you will need templates labeled #footer and/or #actions. The #actions template would be 1 or more buttons while the footer can be anything.

For ActionsAndFooter to be able to inject your templates it requires one parameter in it's constructor. That is a reference to it's parent WidgetContainerComponent.

This is how you would inject that in to your constructor and pass it on:

constructor(@Optional() @Inject(forwardRef(() => WidgetContainerComponent)) protected parent: WidgetContainerComponent){
  super(parent);
}

Here is an example of the templates for each. These go somewhere in your component template:

Actions

<ng-template #actions>
  <button class="dropdown-item my-action">My Action</button>
</ng-template>

As you can see the only requirement is that your button has class of dropdown-item. This is to ensure common styling with the default actions. You should also go to your components scss file and give a class then of 

.my-action {
   :before{
      @include cs-icon($my-cs-icon);
      margin-right: 5px;
   }
}

Footer

<ng-template #footer>
  Your html content here
</ng-template>

So a full html template with actions and footers may look like this:


<div>
  <my-web-part></my-web-part>
</div>
<ng-template #actions>
  <button class="dropdown-item my-action">My Action</button>
</ng-template>
<ng-template #footer>
 My footer
</ng-template>
