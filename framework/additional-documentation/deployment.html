Clearsight Apps Framework

Build An Application

npm install
npm run build-<app-name>

The application name is always lower case and has a dash. So if the package name is Storms.Dashboard then the app name will be storms-dashboard. You can still pass in Storms.Dashboard though and the build will be fine with that. It will however still place the build in dist/apps/storms-dashboard.

Test clearsight-apps

npm install
npm run coverage

This will test all applications and libraries in clearsight-apps repository. It will then generate a coverage report in the coverage folder.

Build All Applications

npm install 
npm run build_all

What should trigger a build?

  1. If a check-in is made to an app (i.e. clearsight-apps/apps/storms-dashboard) that app should be built
  2. If a check-in is made to libs folder (clearsight-apps/libs) then all apps should be built.

Minimum System Requirements

  1. NodeJs >= 10.6.2
